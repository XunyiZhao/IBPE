安装部分：

Python相比于其他编程语言的优势在于它有大量已经编写好的函数库，可以用非常简单的命令实现各种复杂的操作。Anaconda集成了Python中大部分常用的函数库，一键安装后即可方便地使用这些库。下载地址（自己注意操作系统）：
https://www.anaconda.com/download/#windows


启动部分：

安装完anaconda之后，可以采用两种不同的方式编辑python文件。一种是C++课程所接触的脚本式编程，写完一个脚本之后运行。另一种是类似Mathematica的交互式编程，即每次写好几行命令并运行这一段，程序即时输出这一段的运行结果。相对来说，交互式编程更加适合教学目的，因此我们的寒假作业都将采用这种形式的程序。

anaconda已经为我们安装了脚本编辑器spyder，大家也可以自己下载Atom、Vim或sublime text等其他编辑器。对于交互式编程，通常只采用jupyter notebook这一种默认编辑器。编辑完成的文件是.ipynb格式的。下面介绍如何启动：
（参考该链接）http://codingpy.com/article/getting-started-with-jupyter-notebook-part-1/

1、打开目标文件所在的文件夹，按住shift键，空白处点击鼠标右键，选择“打开Powershell”输入jupyter notebook后回车即可。你的浏览器会出现一个新的标签页（浏览器不同可能会有一定差异），之后的编辑就在这样的标签页中进行。


2、找到需要打开的ipynb文件，打开即可。每一段编辑完成后，shift+enter键运行并输出结果。一个已经赋值的变量如果不加任何函数直接运行，将会输出这个函数的值。


3、jupyter中最重要的快捷键就是shift+tab，当我们输入一个函数但不清楚需要提供哪些自变量的时候，可以利用此快捷键调用函数库的docs了解函数的功能和自变量。
